# rz-health

Physical fitness, nutrition, sleep, and mental wellness.

## Scope

Optimize the body and mind as the platform everything else runs on. Without health, no other skill matters. This skill applies the same evidence-based, flywheel approach from the productive skills to physical and mental well-being.

## Workflow

```
BASELINE → PROGRAM → TRAIN → TRACK → ADAPT
```

| Phase | Purpose |
|---|---|
| Baseline | Assess current health metrics — body composition, fitness levels, sleep quality, mental state |
| Program | Design training, nutrition, and recovery protocols based on goals and baseline |
| Train | Execute the program consistently |
| Track | Monitor key health metrics and subjective well-being |
| Adapt | Adjust programming based on progress, plateaus, or life changes |

## Core Domains

- **Physical Training** — strength, cardiovascular fitness, mobility, body composition
- **Nutrition** — macronutrient balance, meal timing, supplementation, hydration
- **Sleep** — sleep hygiene, duration, quality, consistency
- **Mental Wellness** — stress management, meditation, journaling, social connection
- **Recovery** — rest days, deload weeks, injury prevention, active recovery

## Key Principle

> Consistency beats intensity. Train sustainably, eat mostly whole foods, sleep 7-9 hours, and manage stress. The basics done well outperform any optimization hack.
