# Phase: Adapt

## Purpose

Adjust programming based on tracking data, plateaus, life changes, or new goals.

## Triggers

- Progress stalled for 3+ weeks despite consistent execution
- Goal achieved — time for a new target
- Life change (travel, work stress, injury, schedule change)
- Quarterly review indicates program isn't working
- Signs of overtraining (persistent fatigue, declining performance, poor sleep, mood changes)

## Activities

1. **Diagnose** — why is change needed? Data-driven, not feeling-driven
2. **Adjust variables** — change one thing at a time (volume, intensity, frequency, nutrition, sleep)
3. **Update program** — write the new plan; don't just wing it
4. **Set new check-in** — when will you evaluate if the adjustment worked?

## Outputs

- Updated program document
- Log entry explaining what changed and why
- New check-in date

## Completion Criteria

- Change is based on data from Track phase, not impulse
- Only one major variable changed at a time
- New program documented and ready for the next Train cycle
