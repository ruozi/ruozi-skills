# Ruozi Skills (RZ Stack)

A personal skill stack inspired by the G Stack methodology. Covers the full spectrum of capabilities — from productive output to personal growth — organized as structured, actionable skill modules.

## Philosophy

Every skill is a **flywheel**, not a checklist. Each module follows the same structural convention: definition, principles, and phased workflows. The goal is compound growth across all life domains through deliberate practice, reflection, and iteration.

## Folder Convention

Every skill follows the same structure, inherited from the FL Playbook:

```
rz-{skill-name}/
  SKILL.md              # Skill definition, scope, workflow overview
  principles/
    axioms.md           # Core axioms governing the skill
  phases/
    {phase-name}.md     # Detailed phase workflow
```

## Usage

Each skill can be invoked independently or composed. The productive skills reference FL Playbook phases directly — no duplication, single source of truth.
