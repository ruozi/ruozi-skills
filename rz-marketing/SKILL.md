# rz-marketing

Market research — evidence-driven analysis pipeline.

## Scope

Data-driven market research and competitive analysis. Every conclusion must have data or observable action behind it — no subjective speculation allowed.

## Reference

This skill extends **FL Playbook: marketing**. The canonical workflow and phase definitions live in:

```
../fl-playbook/marketing/
```

## Workflow

```
DEFINITION → SOURCE AUDIT → EVIDENCE → ANALYSIS → OUTPUTS → PROPOSAL
```

| Phase | Purpose | FL Playbook Reference |
|---|---|---|
| Definition | Define research goal and objects | `phases/definition.md` |
| Source Audit | Evaluate source quality and reliability | `phases/source-audit.md` |
| Evidence | Collect primary data | `phases/evidence.md` |
| Analysis | Apply analytical methods | `phases/analysis.md` |
| Outputs | Generate reports | `phases/outputs.md` |
| Proposal | Business recommendation | `phases/proposal.md` |

## Axioms (Summary)

Full axioms in `fl-playbook/marketing/principles/axioms.md`:

1. **Growth-Driven Purpose** — research exists to drive growth, not to produce documents
2. **Purpose-First** — know why you're researching before you start
3. **Evidence-Based** — conclusions require evidence with clear hierarchy
4. **Pipeline Dependency** — each stage depends on the previous; no skipping
