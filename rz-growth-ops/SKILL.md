# rz-growth-ops

Growth operations — acquisition, creative production, and ROAS optimization.

## Scope

User acquisition and growth through channel optimization, creative production, and performance marketing. Action-first, ROAS-driven, with creative as the core lever.

## Reference

This skill extends **FL Playbook: growth-ops**. The canonical workflow and phase definitions live in:

```
../fl-playbook/growth-ops/
```

## Workflow

```
STRATEGY → CREATIVE → LAUNCH → MEASURE → OPTIMIZE
```

| Phase | Purpose | FL Playbook Reference |
|---|---|---|
| Strategy | Growth action planning | `phases/strategy.md` |
| Creative | Creative production and testing | `phases/creative.md` |
| Launch | Channel launch and bidding | `phases/launch.md` |
| Measure | Attribution and ROAS calculation | `phases/measure.md` |
| Optimize | Budget allocation decisions | `phases/optimize.md` |

## Axioms (Summary)

Full axioms in `fl-playbook/growth-ops/principles/axioms.md`:

1. **Action-Driven** — imperfect execution beats perfect plans
2. **ROAS-Driven** — every dollar must justify itself
3. **Creative is the Lever** — creative-targeting symbiosis drives performance
4. **Always-On Experimentation** — two-layer expectations (base + test)
5. **Creative Fatigue Management** — track and refresh before decay
