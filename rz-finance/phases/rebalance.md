# Phase: Rebalance

## Purpose

Adjust portfolio and strategy based on monitoring data and life changes. This is where discipline meets adaptability.

## Triggers

- Portfolio drift exceeds threshold (typically 5% from target allocation)
- Major life event (job change, marriage, child, relocation)
- Goal achieved or timeline changed
- Tax-loss harvesting opportunity
- Market regime change requiring strategic review

## Activities

1. **Rebalance portfolio** — sell overweight, buy underweight to return to target allocation
2. **Update goals** — adjust targets based on life changes
3. **Revise allocation** — if risk profile has genuinely changed, update target allocation
4. **Harvest losses** — sell losing positions for tax benefit; reinvest in similar (not identical) assets
5. **Review and update automation** — adjust contribution amounts if income changed

## Outputs

- Rebalanced portfolio (within threshold of target)
- Updated financial plan (if life changes warrant)
- Tax-loss harvesting log
- Notes on what triggered the rebalance and actions taken

## Completion Criteria

- Portfolio within acceptable range of target allocation
- All triggered actions completed
- Lessons captured for future monitoring cycles
