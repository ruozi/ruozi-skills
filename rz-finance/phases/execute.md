# Phase: Execute

## Purpose

Deploy capital according to plan. Automate wherever possible to remove emotion and friction.

## Activities

1. **Automate contributions** — set up automatic transfers to investment and savings accounts
2. **Execute trades** — buy target allocations; use limit orders for large positions
3. **Pay down debt** — execute debt strategy from Plan phase
4. **Fund emergency reserve** — priority until target reached
5. **Optimize income** — negotiate raises, develop side income, increase earning capacity

## Outputs

- Automated contribution schedules active
- Portfolio positions match target allocation
- Debt payments on schedule
- Emergency fund progress tracked

## Completion Criteria

- All planned contributions are automated and running
- Portfolio is within acceptable drift from target allocation
- No unplanned discretionary spending disrupting the plan
